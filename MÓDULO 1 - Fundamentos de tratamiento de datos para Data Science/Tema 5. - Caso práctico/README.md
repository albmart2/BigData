# Se pide

## Enunciado

En este ejercicio, se van a poner en práctica algunas de las funciones que ofrece GitHub.

## Datos

- Buscar un repositorio GitHub cuyas palabras de búsqueda sean "BigData", "Jupyter Notebook".
- Crear una copia del repositorio elegido en la cuenta propia.
- Añadir un archivo de texto nuevo e introducir el mensaje "Ejemplo de archivo".

# Solución

1. Usando el buscador de GitHub, se introducen las palabras claves "Big Data".
2. En los resultados obtenidos, se filtra por “Jupyter Notebook”.
3. En los resultados obtenidos, se selecciona uno de los repositorios.
4. Una vez dentro del repositorio seleccionado, se pulsa sobre <i>Fork</i> para crear la copia.
5. Se crea la copia en el usuario.
6. Por último, para crear un archivo, se pulsa sobre <i>Create new file</i>.
7. Se añade el título y el contenido y se pulsa sobre <i>Commit new file</i>.
