# Caso práctico
## Enunciado

En esta segunda unidad, y debido a su carácter teórico e introductorio, el caso práctico se basará en la identificación de una herramienta de <i>modern BI</i> y en definir cuáles son sus funcionalidades clave. El objetivo de este caso práctico es asentar los conceptos vistos en esta unidad y así <b>afianzar los conocimientos adquiridos</b>.

## Se pide

Para este caso práctico, se propone la tarea de identificar cuáles son las características clave de una herramienta del área del modern BI a elección del alumnado.

<b>Si no se tiene acceso a una herramienta de modern BI, se recomienda realizar la prueba con DOMO, que es una herramienta bien documentada y con un trial de 30 días.</b>

La solución se redactará con el mayor detalle que se pueda, siguiendo los siguientes puntos:

1. <b>Motivo por el que se ha seleccionado la herramienta.</b>
2. <b>Identificar las siguientes funcionalidades</b>:
    - <b>Autoservicio.</b>
    - <b>Seguridad.</b>
    - <b>Data discovery y visualizaciones.</b>
    - <b>Análisis predictivo y prescriptivo.</b>
    - <b>Inteligencia de negocio colaborativa.</b>
    - <b>BI móvil.</b>

## Solución
1. <b>Motivo por el que se ha seleccionado la herramienta.</b>

    Se ha seleccionado DOMO por ser una herramienta de BI que dispone de un sistema totalmente cloud.

    Es decir, incluye desde las visualizaciones, ETL con módulos de Python y R, almacén de datos hasta gestión de seguridad integrado en una misma plataforma. Es uno de los principales referentes del modern BI, aunque su distribución nacional todavía no es muy grande.

2. <b>Identificar las siguientes funcionalidades</b>:
    - <b>Autoservicio.</b> Paso de datos en bruto a visualizaciones de forma muy sencilla. Permite navegar para profundizar en los datos y establecer un framework para establecer el autoservicio.

      <img width="1300" height="812" alt="image" src="https://github.com/user-attachments/assets/dafc2b63-8887-4d44-89fb-62fb4d2a641a" />

    - <b>Seguridad.</b>
   
        Domo permite seguridad en el contenido, columnas y filas. Es decir, se puede configurar el acceso de los usuarios a los datos de forma muy flexible.

        Además, al estar en la nube cumple y está validada por varios protocolos de seguridad y encriptaciones.

        <img width="383" height="226" alt="image" src="https://github.com/user-attachments/assets/b8dbbaad-3e90-491d-9241-4d27d93b3b60" />

    - <b><i>Data discovery</i> y visualizaciones.</b>

        Dispone de las visualizaciones más generales, así como de un grupo de opciones que permiten filtrar la información y navegar para ver los datos en mayor detalle.

        <img width="1210" height="575" alt="image" src="https://github.com/user-attachments/assets/93962004-24f8-4a46-a79b-9060d70f9879" />

    - <b>Inteligencia artificial.</b>

        Como muchas herramientas, ya dispone de un análisis automático de los datasets que se usan en las visualizaciones para presentar análisis y correlaciones de los datos que puedan arrojar nueva información para el usuario. En este caso se llama “<b>Ideas</b>”.

        <img width="715" height="318" alt="image" src="https://github.com/user-attachments/assets/d1eaff1b-b274-4bbe-a1af-b14b6aed7ba6" />

    - <b>Análisis predictivo y prescriptivo.</b>

        Domo permite combinar objetos arrastrando y soltando con los joins y agrupaciones de datos ya simplificados en ETL, con seis acciones de ciencia de datos:

        - Clasificación.
        - Pronóstico.
        - Detección de valores atípicos.
        - Valores atípicos multivariados.
        - Clustering.
        - Predicción.
     
        <img width="1374" height="680" alt="image" src="https://github.com/user-attachments/assets/69863d5f-b4b7-428a-8e2b-4682486c33bb" />

    - <b>Inteligencia de negocio colaborativa.</b>

        Domo presenta los cambios en sus datos en tiempo real y se pueden comunicar al resto del equipo directamente dentro de Domo para discutir, asignar tareas y tomar medidas.

        Mantiene el equipo conectado. Con perfiles y organigramas, se puede ver dónde encajan las personas, conectarse y aprender cómo contribuyen a la organización.

        Domo permite estar actualizado sobre las conversaciones, recibiendo las últimas actualizaciones en un dispositivo mediante alertas en tiempo real.

        La herramienta de chat de Domo, Buzz, está integrada de forma nativa en la plataforma Domo para garantizar que pueda colaborar fácilmente en los datos en tiempo real que proporciona Domo.

        <img width="1212" height="676" alt="image" src="https://github.com/user-attachments/assets/5865cb3a-f8da-4242-9ad7-b56fc18707c9" />

    - <b>BI móvil.</b>

        Domo usa una plataforma única para enviar actualizaciones a dispositivos móviles de forma instantánea. La aplicación móvil mantiene los datos actualizados automáticamente. No requiere desarrollo adicional.

        Permite colaboración con todo el equipo, todos los datos se reúnen y son accesibles para su equipo, estén donde estén, solo se necesita acceso a internet.

        Los datos siempre están protegidos, ya que incluye autenticación de aplicaciones móviles, notificaciones push cifradas, etc.
