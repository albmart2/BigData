# Caso práctico

## Enunciado

Este ejercicio debe realizarse con la herramienta Tableau.

Lo primero que debe hacer el alumno es descargar el archivo que se utilizará como fuente de datos en el siguiente enlace: [Global_Superstore.xls](https://github.com/user-attachments/files/27288168/Global_Superstore.xls)

## Se pide
El alumno debe imaginar que trabaja en el departamento de análisis de riesgo de una multinacional cuya actividad principal es la fabricación y distribución de todo tipo de productos.

Debido a que el Consejo de Administración está preocupado por la evolución de los márgenes, en los diferentes departamentos, le piden un análisis que permita identificar los problemas subyacentes para poder implantar acciones correctivas que mejoren la situación actual de la compañía.

Asimismo, los responsables comentan que es posible que, en algunas de las líneas de trabajo, no se esté obteniendo el beneficio esperado.

Para realizar el análisis, el departamento financiero y de sistemas le entregan un Excel con toda la información (Global Superstore.xls), que el alumno debe usar como fuente de datos.

<b>Se recomienda visualizar los webinars de Tableau incluidos en la unidad antes de realizar esta práctica, ya que en él se explican en detalle los cuadros de mando e historias.</b>

<b>Específicamente, se deben generar las siguientes visualizaciones (hojas):</b>

1. Ventas por mercado, categoría y subcategoría de productos, detallando el beneficio.
2. Mapa de ventas y beneficio.
3. Serie temporal del beneficio, por año, para detectar tendencias.
4. Correlación entre el coste de envío y el beneficio, indicando la tendencia.
5. Crear un cuadro de mando.
6. Crear una historia.
